# Cucumber Export Sales & Defect Analysis Dashboard

## Project Overview
This project focuses on analyzing the sales and defect data of a cucumber export company using Power BI. The objective was to clean, analyze, and visualize the dataset to identify sales performance trends, defect patterns, and generate meaningful business insights.

## Tools & Technologies Used
- Power BI
- Microsoft Excel
- Data Cleaning & Preprocessing
- Data Visualization
- Data Analysis

## Dataset Description
The dataset contains information related to cucumber export operations, including:
- Sales details
- Product information
- Defect information
- Quality-related data

The dataset was prepared and cleaned before performing analysis and visualization.

## Project Workflow

### 1. Data Cleaning
- Removed unnecessary and inconsistent data.
- Handled missing values and formatting issues.
- Prepared the dataset for accurate analysis.

### 2. Data Analysis
- Analyzed sales performance and defect information.
- Identified patterns and trends from the dataset.
- Compared sales data with defect occurrences.

### 3. Dashboard Development
Created an interactive Power BI dashboard containing:
- Sales analysis visuals
- Defect analysis visuals
- Key performance indicators (KPIs)
- Data-driven insights for decision making

## Dashboard Features
- Sales performance overview
- Defect tracking and analysis
- Interactive filters and visualizations
- Business insights through graphical representation
